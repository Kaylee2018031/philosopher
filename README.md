# 철학자 인용 카드 웹사이트

철학자 이름, 책 제목, 원문 인용, 한글 번역, 짧은 해설, 인용 정보를 카드로 관리하는 작은 웹 앱입니다.

## 기능

- 데리다, 푸코, 한나 아렌트, 시몬 베유 샘플 카드 제공
- 새 인용 카드 추가 폼
- 검색어로 철학자, 책, 인용문, 번역, 해설, 인용 정보 필터링
- `localStorage` 저장으로 새로고침 후에도 사용자가 추가한 카드 유지
- 모바일 친화적인 반응형 레이아웃

## 실행

별도 빌드 과정 없이 `index.html`을 브라우저에서 열면 됩니다.

로컬 서버로 확인하려면 다음 명령을 사용할 수 있습니다.

```bash
python3 -m http.server 8000
```

그 다음 브라우저에서 <http://localhost:8000>을 엽니다.
