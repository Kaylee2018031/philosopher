# 데리다: 책 구절, 영어 원문, 한글 번역

## Jacques Derrida, *Of Grammatology* (1967; English trans. 1976)

> “There is nothing outside of the text.”

**한글 번역:** “텍스트 바깥에는 아무것도 없다.”

**프랑스어 원문 참고:** “Il n’y a pas de hors-texte.”

## 짧은 해설

이 구절은 흔히 “현실은 존재하지 않고 오직 글만 있다”는 뜻으로 오해되지만, 데리다가 강조한 핵심은 의미가 언제나 언어, 기호, 맥락, 해석의 관계망 안에서 구성된다는 점입니다. 따라서 어떤 의미도 완전히 순수한 ‘바깥’에서, 즉 맥락과 매개 없이 직접 주어지지 않는다는 주장으로 읽는 편이 적절합니다.

## 인용 정보

- 저자: Jacques Derrida
- 책: *Of Grammatology*
- 프랑스어 초판: 1967년
- 영어 번역: Gayatri Chakravorty Spivak, 1976년
- 구절 위치: 루소를 읽는 2부 논의에서 자주 인용되는 문장

> 참고: 위 영어 문장은 널리 알려진 영어 번역문입니다. 데리다의 실제 원문은 프랑스어이며, 더 직역하면 “바깥-텍스트는 없다” 또는 “텍스트의 바깥은 없다”에 가깝습니다.
